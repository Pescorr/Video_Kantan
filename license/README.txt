== Video Kantan - License Activation ==
== Video Kantan - ライセンス有効化 ==

[English]
To activate the full version (remove watermark from exports):

1. Purchase a license key
2. Place the received "license.key" file in this folder
3. Restart Video Kantan

The watermark on exported videos/GIFs will be removed.

[日本語]
フルバージョンを有効にするには（出力のウォーターマークを解除）:

1. ライセンスキーを購入してください
2. 受け取った「license.key」ファイルをこのフォルダに配置してください
3. Video Kantan を再起動してください

出力される動画/GIF のウォーターマークが解除されます。
